# 钉钉 DM 表格渲染方案

## 问题

Hermes 适配层 → 钉钉 DM 的 Markdown 管道表格（`|col|col|`）渲染不稳定。2026-06-06 会话中用户多次反馈格式不对（「表格的渲染还是不正确」「你现在格式的 md的渲染，有问题」）。

根因：Hermes 消息通道与钉钉原生 IM API 的 Markdown 渲染引擎不同。

## 解决方案：dws chat message send

用 `dws chat message send` 走钉钉原生 IM API 发送含表格的消息，绕过 Hermes 适配层。

```bash
dws chat message send --user 2267566123688378 \
  --title "审批列表" \
  --text "## 审批列表
| # | 申请人 | 类型 | 金额 |
|---:|---|---:|---|
| 1 | 郭郡 | 差旅报销 | ¥480 |" \
  --format json
```

## 使用规则

- 含表格内容 → dws 发送
- 纯文本确认/简短回复 → Hermes 正常回复
- dws 发送后 Hermes 只回简短确认，避免内容重复
- 表格 ≤5 列，超长分批发送

## 备用方案（dws 不可用时）

emoji 分层 + 纯文本编号列表：

