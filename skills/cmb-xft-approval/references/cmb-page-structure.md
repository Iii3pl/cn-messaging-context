## 登录检测
- 已登录: title 包含「智能费控」（如 `智能费控·薪福通`）
- 未登录/过期: title 为 `招商银行`（不含智能费控），或弹窗「长时间未操作，请重新登录」
- 服务端 30 分钟无操作自动踢人

## 登录流程

```
首页 → 点击「登录」(a[href*="/#/index"])
     → 切换到「密码登录」tab (rc-tabs-0-tab-password)
     → 输入手机号 (#passwordLogin_phone)
     → 输入密码 (#passwordLogin_password)
     → 点击登录按钮
     → ⚠️ 滑块验证码（只能人工拖动）
     → 进入 /#/workbench
     → 点击「智能费控」应用卡片 (FullApplication_item__3Ruc3)
     → TripMainWeb 加载
```

### 字段提取规则（新版）

| 字段 | 正则 | 示例 |
|------|------|------|
| 申请人+工号 | `(\S+)\s*[-–]\s*(\d{6})\s*[-–]\s*\S+` | 施璐璐 - 000806 - 公司 |
| 金额 | `/报销金额[（(]CNY[）)]\s*([\d,.]+)/` | CNY 880.32 |

## 审批按钮（关键！两步提交）

按钮位于页面底部，需 `scroll('down', 5000)` 后才可见。

### ⚠️ 必须点击 `<button>` 本身，不能点内部 `<span>`
```
<button class="ant-btn ant-btn-primary" onclick=true>  ← 点这个！
  <span>通过</span>                                   ← 不能点这个
</button>
```

### 两步提交
1. 点击「通过」(`button.ant-btn-primary`) → 选中动作
2. 点击「确认」(`button.guideStepOperateOkButton`) → 真正提交

`button.click()` 直接有效，不需要 Vue 实例。
